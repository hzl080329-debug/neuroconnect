# 开发需求文档

## 项目概述
**项目名称**：神经多样性人群匿名互助社区（暂定）

**项目定位**：类似 Reddit 的匿名互助社区网站，面向 ADHD、自闭谱系、焦虑、抑郁等神经多样性及精神健康人群。提供匿名、安全、包容的环境，帮助用户分享就诊经历、发布日常帖子、互相交流、提问求助。

**核心问题**：神经多样性人群在就医、诊断、用药、生活适应过程中缺乏真实经验参考和安全无评判的交流空间。

---

## 目标用户
- ADHD（注意力缺陷多动障碍）人群
- 自闭谱系人群
- 焦虑症、抑郁症患者
- 其他神经多样性及精神健康相关人群
- 关注精神健康的家属、朋友、专业人士

---

## 功能需求

### F1. 用户系统
- F1.1 邮箱或手机号注册（不公开真实信息）
- F1.2 设置匿名昵称（唯一，用于社区展示）
- F1.3 选择预设匿名头像（不允许上传真实头像，保护隐私）
- F1.4 编辑个人简介
- F1.5 登录/登出
- F1.6 个人主页（展示公开信息、历史帖子、收藏等内容）

### F2. 内容发布
- F2.1 发布帖子：选择社区板块、标题、正文、标签、匿名身份
- F2.2 帖子支持文本内容（第一阶段不做富文本，Markdown 即可）
- F2.3 编辑自己发布的帖子
- F2.4 删除自己发布的帖子

### F3. 社区板块
- F3.1 ADHD — 注意力缺陷多动障碍相关讨论
- F3.2 自闭谱系 — 自闭症谱系相关讨论
- F3.3 焦虑抑郁 — 焦虑、抑郁等情绪障碍讨论
- F3.4 就诊经历 — 医院就诊、诊断过程分享
- F3.5 生活分享 — 日常生活、兴趣爱好
- F3.6 学习工作 — 学习技巧、工作适应
- F3.7 情绪支持 — 情绪倾诉、互相支持

### F4. 互动功能
- F4.1 帖子评论（支持回复评论，形成讨论串）
- F4.2 点赞帖子
- F4.3 收藏帖子
- F4.4 分享帖子链接

### F5. 搜索与发现
- F5.1 关键词搜索帖子
- F5.2 标签筛选
- F5.3 热门帖子排序
- F5.4 最新帖子排序
- F5.5 首页推荐板块

### F6. 就诊经历分享
- F6.1 结构化表单填写（医院名称、科室、诊断结果、用药、费用、评分、体验描述）
- F6.2 就诊经历列表浏览
- F6.3 就诊经历详情查看

### F7. 匿名问答区
- F7.1 匿名提问
- F7.2 匿名回答
- F7.3 问题和答案的点赞

### F8. 私信功能
- F8.1 用户一对一异步私信
- F8.2 私信列表
- F8.3 私信对话页
- F8.4 未读消息提醒

### F9. 社区治理
- F9.1 举报帖子/评论（选择举报原因）
- F9.2 举报达阈值（3次）自动隐藏内容
- F9.3 管理员后台查看举报列表
- F9.4 管理员审核：恢复或删除内容
- F9.5 社区规则页面

### F10. 资源中心
- F10.1 科普文章（管理员发布）
- F10.2 就诊指南（管理员发布）
- F10.3 求助热线列表
- F10.4 外部支持资源链接

### F11. 通知系统
- F11.1 评论回复通知
- F11.2 点赞通知
- F11.3 私信通知
- F11.4 举报处理结果通知

## 非功能需求

### N1. 国际化
- 界面支持中文和英文切换
- 所有 UI 文案双语化
- 用户内容不做翻译，维持原语言

### N2. 响应式设计
- 电脑浏览器（≥1024px）
- 平板浏览器（768-1023px）
- 手机浏览器（320-767px）

### N3. 无障碍（WCAG 2.1 AA）
- 颜色对比度 ≥ 4.5:1
- 最小字号 16px
- 支持键盘导航
- 所有交互元素有 focus 可见指示
- 图片有 alt 文本
- 尊重 prefers-reduced-motion

### N4. 安全性
- 用户真实信息（邮箱/手机）不公开
- 匿名昵称和真实身份无关联
- 内容举报和审核机制
- 防止 SQL 注入、XSS 攻击

### N5. 性能
- 首页加载时间 < 3秒
- 页面切换流畅
- 图片懒加载

### N6. 可扩展性
- 架构支持未来扩展为 PWA
- 数据库设计预留扩展空间
- API 设计为 App 调用做好准备

- 中英双语

- 富文本编辑器
- 更丰富的搜索

### v2.0 — 扩展
